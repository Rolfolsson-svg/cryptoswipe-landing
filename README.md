# CryptoSwipe Landing

En färdig Next.js + Tailwind landningssida med swipe-demo.

## Kör lokalt
```bash
npm install
npm run dev
# öppna http://localhost:3000
```

## Deploy till Vercel
- Skapa ett GitHub-repo och pusha koden
- Importera på https://vercel.com/new
- Framework: Next.js, Build: `npm run build`
